maven
=====

个人maven仓库托管

# 使用方法

在pom文件中加入下面信息：

```xml
<repositories>
	<repository>
		<id>gfs-maven-snapshot-repository</id>
		<name>gfs-maven-snapshot-repository</name>
		<url>https://raw.githubusercontent.com/gefangshuai/maven/master/</url>
	</repository>
</repositories>
```
